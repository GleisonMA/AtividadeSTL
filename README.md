# AtividadeSTL

Atividade referente à disciplina de desenvolvimento orientado à objeto em C++ para implementação de exemplos de utilização dos algoritimos genéricos da biblioteca padrão.

## Exemplos a serem implementados

- fill_n
- equal
- mismatch
- remove_copy_if
- replace_copy_if
- inplace_merge
- reverse_copy
- find_if_off
- search_n
- for_each
- transform
- nth_element
- count_if
- make_heap
- times
- greater
- greater_equal
- less_equal
- logical_and
- logical_or
- logical_not
- men_fun_ref
